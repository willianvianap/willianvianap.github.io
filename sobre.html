SOBRE ESTA PÁGINA

Um pouco do processo de criação desta página.

Sobre:


Este projeto foi criado para servir como uma página de perfil simples e moderna, reunindo meus principais links em um só lugar. Além de facilitar o acesso às minhas redes e projetos, ele também representa minha evolução nos estudos de desenvolvimento web.

Processo:


O desenvolvimento foi feito utilizando apenas HTML e CSS, aplicando os conhecimentos adquiridos durante meus estudos. Ao longo da criação, trabalhei em detalhes como alinhamento, estilização, efeitos de hover, organização do código e responsividade, usando este projeto como forma de praticar e aprimorar minhas habilidades.
